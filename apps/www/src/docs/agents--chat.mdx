# Chat Agent
Chat agents are the basic building blocks, everything else just extends them.

**Agent Definition**
<span>
  `@/ai/agents/coding-assistant.ts`
</span>

```ts
import { createChatAgent } from "@/agents"

const primaryIdentity = `
  You are an expert coding assisstant AI, tasked with helping users build sick apps.
`

export const codingAssistantAgent = createChatAgent({
  config: {
    model: "gpt-4",
    max_tokens: 500,
    temperature: 0.2
  },
  identityMessages: [
    {
      role: "system",
      content: primaryIdentity
    }
  ]
})
```
