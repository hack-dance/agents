
# Installation


## Existing projects
Install the necessary dependencies with your package manager of choice.
  ```bash
    pnpm add openai zod
  ```

Add the core agent utilities
  ```zsh
    npx @hackdance/agents add --path=src
  ```


Optionally setup import aliases
  ```json
    {
      "compilerOptions": {
        "paths": {
          "@/*": ["./*"]
        }
      }
    }
  ```
